## 关于本书

本书面向数据结构与算法初学者，致力于达成以下目标：

- 开源免费，所有同学都可在网上获取本书；
- 新手友好，适合算法初学者自主学习入门；
- 动画讲解，尽可能地保证平滑的学习曲线；
- 代码导向，提供可一键运行的算法源代码；
- 讨论学习，提问一般能在三日内得到回复；

如果感觉本书对你有所帮助，请点个 Star :star: 支持一下，谢谢！
